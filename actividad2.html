Código: 


            #include <stdio.h>

            int main() {
                int edad;
                printf("¿Cuántos años tienes? ");
                scanf("%d", &edad);
            if (edad >= 18) {
                printf("Eres mayor de edad.\n");
            } else {
                printf("Eres menor de edad.\n");
            }
            return 0;
            }

Explicación

#include <stdio.h> Añade las bibliotecas de C.

int main() {} Es la función principal de C, "int" indica que devuelve un entero al sistema operativo
, normalmente 0 para destacar que termino correctamente.

int edad; Genera una variable de un numero entero llamada edad.

scanf("%d", &edad); Una función para leer datos desde la entrada estándar
Entre comillas está la máscara del tipo de dato que se requiere, en este caso %d es un entero
Finalmente con & en la variable edad.

if (edad >= 18) {
                printf("Eres mayor de edad.\n");
            } else {
                printf("Eres menor de edad.\n");
            }
Es una estructura condicional, si la variable edad es mayor o igual a 18, muestra en pantalla con printf "Eres mayor de edad" y un salto de linea con /n
Si es menor de 18 muestra de igual forma "Eres menor de edad" con su salto de linea.

return 0; Regresa 0 al sistema operativo, indicando que el programa terminó.
